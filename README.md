# Trade Simulator

## Overview

My ramblings on various stock analysis and data mining.

## Prerequisites

- Python 2.7
- NumPy (<http://www.numpy.org>)
- SciPy (<http://www.scipy.org>)
- Pandas (<http://pandas.pydata.org/>)
- mpl_finance (<https://github.com/matplotlib/mpl_finance>)
